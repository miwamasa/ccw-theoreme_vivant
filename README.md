# Cédric Villani フィールズ賞業績 インタラクティブ学習アプリ

セドリック・ヴィラーニ氏の2010年フィールズ賞受賞業績について、数学・物理学・機械学習の3つの視点から多角的に学べるインタラクティブなWebアプリケーションです。

## 概要

このアプリケーションは、ヴィラーニ氏の主要な研究テーマである以下の内容を、わかりやすく、インタラクティブに学習できるように設計されています：

- **ボルツマン方程式**の解析
- **ランダウ方程式**と非平衡統計物理学
- **エントロピー増大**と**準統御性（Hypocoercivity）**
- **最適輸送理論**と**ワッサースタイン距離**
- **非線形ランダウ減衰**の証明

## 主な機能

### 1. 3つの視点からの学習

- **数学的視点**: 偏微分方程式、関数解析、最適輸送理論の数学的基礎
- **物理学的視点**: 統計物理学、非平衡現象、プラズマ物理学への応用
- **機械学習的視点**: 現代のAI・機械学習における最適輸送理論の応用

### 2. インタラクティブな要素

- **タブ切り替え**: 3つの視点を簡単に切り替えて学習
- **展開可能なセクション**: トピックごとに内容を展開・折りたたみ可能
- **理解度チェッククイズ**: 重要な概念を確認するクイズ
- **可視化とアニメーション**:
  - 最適輸送の視覚的なアニメーション
  - エントロピー減衰のリアルタイムシミュレーション
- **数式表示**: KaTeXによる美しい数式レンダリング

### 3. レスポンシブデザイン

デスクトップ、タブレット、スマートフォンなど、あらゆるデバイスで快適に閲覧できます。

## 使い方

### 起動方法

1. このリポジトリをクローンまたはダウンロードします：
   ```bash
   git clone <リポジトリURL>
   cd ccw-theoreme_vivant
   ```

2. `index.html` をWebブラウザで開きます：
   - ファイルをダブルクリック、または
   - ブラウザにドラッグ&ドロップ、または
   - ローカルサーバーを起動（推奨）：
     ```bash
     # Pythonを使用する場合
     python -m http.server 8000
     # または
     python3 -m http.server 8000

     # Node.jsのhttp-serverを使用する場合
     npx http-server
     ```
   - ブラウザで `http://localhost:8000` にアクセス

### 操作方法

1. **タブの切り替え**
   - 上部のナビゲーションバーから「数学的視点」「物理学的視点」「機械学習的視点」を選択

2. **トピックの展開**
   - 各セクションのタイトルをクリックすると、内容が展開・折りたたみされます

3. **クイズに挑戦**
   - 各視点に配置されたクイズで理解度をチェック
   - 選択肢をクリックすると、即座にフィードバックが表示されます

4. **可視化とシミュレーション**
   - **最適輸送のアニメーション**: 「輸送をアニメーション」ボタンで開始
   - **エントロピー減衰シミュレーション**:
     - スライダーで衝突頻度を調整
     - 「シミュレーション開始」ボタンでグラフが描画されます

## 学習のヒント

1. **順番に学習する**: まず「数学的視点」から始めて、基礎的な概念を理解してから他の視点に進むことをお勧めします。

2. **クイズで確認**: 各セクションを読んだ後、クイズで理解度を確認しましょう。

3. **可視化を活用**: 抽象的な概念は、アニメーションやシミュレーションで視覚的に理解すると効果的です。

4. **複数の視点を比較**: 同じ概念が異なる視点でどう説明されるかを比較することで、より深い理解が得られます。

5. **参考資料を読む**: `relateddocs/` ディレクトリには、より詳細な解説が含まれています。

## トラブルシューティング

### 数式が表示されない
- インターネット接続を確認してください（KaTeXはCDN経由で読み込まれます）
- ブラウザのキャッシュをクリアして再読み込みしてください

### アニメーションが動かない
- JavaScriptが有効になっているか確認してください
- ブラウザのコンソールでエラーを確認してください

### スタイルが適用されない
- CSSファイルのパスが正しいか確認してください
- ブラウザの開発者ツールでネットワークエラーを確認してください
